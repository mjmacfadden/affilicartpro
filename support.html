Installation

Prerequisites: You'll need an active WordPress site running WordPress 5.0 or higher. Affilicart also works best with a custom domain (not WordPress.com).

Step 1: Install Affilicart

You can install Affilicart directly from your WordPress admin panel (easiest method) or manually upload it.

Option A: Install from WordPress Admin Dashboard (Recommended)

  1. Log in to your WordPress admin dashboard
  2. Go to Plugins → Add New
  3. Search for "Affilicart"
  4. Click Install Now on the Affilicart plugin
  5. Click Activate

Option B: Manual Upload

  1. Download Affilicart from the WordPress Plugin Directory or affilicartpro.com
  2. Log in to your WordPress admin dashboard
  3. Go to Plugins → Add New → Upload Plugin
  4. Choose the affilicart.zip file and click Install Now
  5. Click Activate Plugin

Step 2: Verify Installation

Once activated, you should see "Affilicart" appear in your WordPress admin menu.

✔ Installation Complete! You're ready to create your first affiliate cart.

Quick Start Guide

Affilicart is designed to be simple. Here's what you need to know:

1. Set Up Your Amazon Affiliate ID

Before adding products, configure your Amazon Associates affiliate ID:

What is an Affiliate Tag?

Your Amazon Associates ID (affiliate tag) is how Amazon tracks sales from your recommendation and credits your commission. Every product you add to Affilicart uses this tag automatically.

Finding Your Amazon Associates ID

  1. Log in to affiliate-program.amazon.com
  2. Go to Account Settings
  3. Under Tracking IDs, copy your ID (e.g., "mysite-20")

Add Your ID to Affilicart

  1. Go to Affilicart → Settings in your WordPress admin
  2. Paste your Amazon Associates ID into the settings
  3. Click Save Settings

Verification

To verify your tag is working:

  1. Create a test product with an Amazon link
  2. Add it to the cart on the front end, click the shopping cart icon to open the modal
  3. Click "Checkout on Amazon"
  4. View the URL to ensure your affiliate ID is appended correctly

2. Add Products

Go to Affilicart → Add Product to create product entries for your cart.

Fill in the following fields:

  • Product Name — The name of the product (e.g., "Sony WH-1000XM5 Headphones")
  • Product Description — A brief description of why you recommend this product
  • ASIN — The Amazon Standard Identification Number. You can find this on any Amazon product page in the product details section, or in the URL (e.g., for "amazon.com/dp/B0XXXXX", the ASIN is B0XXXXX)
  • Product Image — Upload an image of the product. ⚠️ Important: You must own the right to use this image. Using Amazon's product images without permission is a violation of their Terms of Service. Use product images from the manufacturer, take your own photos, or use licensed images.
  • Product Category — Select the category this product belongs to (e.g., Electronics, Books, Home & Kitchen)

Click Save when you're done. Repeat this process for each product you want to add to your cart.

3. Display Products Using Shortcodes

Add products to any page or post using these shortcodes:

Use Case Shortcode
Display all products [affilicart_grid]
Display a single product [affilicart_grid id="123"]
Display multiple products [affilicart_grid id="123,456,789"]

Customize Product Display

Hide specific fields by adding parameters to your shortcode:

To Hide Add This Parameter
Product Image show-image="no"
Product Title show-title="no"
Product Description show-description="no"
Product Price show-price="no"

Example:

[affilicart_grid id="123" show-description="no"]

This displays product 123 without the description.

Display a Single Button

Add a standalone "Add to Cart" button for a specific product anywhere on your site:

[affilicart_button id="123"]

Use Case: Perfect for embedding product buttons within blog posts, sidebars, or anywhere you want a minimal call-to-action without the full product card.

Display a Link with Hover Card

Add a clickable text link that shows a product preview (image, name, price) on hover:

[affilicart_link id="123"]

This displays the product name as a link. When you hover over it, a small card appears showing:

  • Product image
  • Product name
  • Add to Cart button

Custom Link Text: You can change the link text by adding a text parameter:

[affilicart_link id="123" text="View Details"]

Use Case: Embed product links naturally within blog post text, reviews, or recommendations. Readers can preview the product without leaving your page.

🎯 Pro Tips

  • Find Product IDs: Go to Products list — the ID is shown under the product name
  • Custom Styling: Use the Accent Color setting to match your site theme without code changes
  • High-Res Images: The lightbox automatically loads full-resolution product images for better viewing
  • Shopping Cart: All cart data is saved in the browser and persists across page visits until cleared

Auto-Tagging

Affilicart automatically appends your Amazon affiliate ID to all Amazon links in your products—including regular product links and shortened Amazon links. This ensures that commissions are properly attributed to you.

How it works:

  • Any Amazon link (regular or shortened) without an affiliate tag will automatically get your ID appended
  • If a link already contains someone else's affiliate ID, Affilicart will not override it
  • If your ID is already present in the link, Affilicart won't add it twice — it simply leaves the link unchanged

Example:

  • https://amazon.com/dp/B0XXXXX → Gets your ID added automatically
  • https://a.co/d/0dW3D1Vx (shortened link) → Gets your ID added automatically
  • https://amazon.com/dp/B0XXXXX?tag=other-id-20 → ID is not replaced (respects existing attribution)
  • https://amazon.com/dp/B0XXXXX?tag=your-id-20 → Already has your ID, no changes

This feature allows you to confidently add any Amazon product link, knowing that Affilicart will ensure proper commission tracking when the link doesn't already have another affiliate tag. Additionally, if you previously shared a link that lacked tracking information, Affilicart will now credit you for future sales from those historical links.

Amazon API Live Pricing (Beta)

Affilicart is currently developing live Amazon pricing integration to display real-time product prices directly in your carts. This feature is currently in beta and experimental.

Current Status

  • In active development and testing
  • May not work for all users or all products
  • Pricing data may occasionally be unavailable or inaccurate
  • Updates and improvements are ongoing

Important Notes

If you encounter issues with live pricing, you can manually input product prices in your cart settings. For now, many users prefer to let readers see current pricing directly on Amazon's checkout page rather than displaying potentially stale prices on your site.

Frequently Asked Questions

Do I need an Amazon Associates account?

Yes. Affilicart works with your Amazon Associates ID to track and attribute sales. If you don't have an account, sign up here.

Is there a limit to how many products I can add?

No! Create as many products as you need with unlimited carts and customization.

Can I use Affilicart with other affiliate programs?

Currently, Affilicart only supports Amazon Associates. We're exploring other integrations for future releases.

What happens when a reader clicks "Checkout on Amazon"?

Readers are taken to Amazon.com with all your recommended products pre-loaded in their cart. Your affiliate link is automatically included.

Do I earn commission on all items in the cart?

Amazon's standard commission structure applie to all items in a user's cart.

Why doesn't Affilicart display live product prices?

Displaying static or outdated prices violates Amazon's Terms of Service. Affilicart is developing live Amazon API pricing integration, but Amazon API access is only granted to users who meet their qualifying criteria: a minimum of 10 sales per month generated through your affiliate links in the previous month.

Until you reach this threshold, Affilicart lets readers see current pricing directly on Amazon's checkout page when they click to purchase. This ensures prices are always accurate and compliant with Amazon's policies.

Is there a character limit for cart titles or product descriptions?

No strict limits, but keep titles under 100 characters and descriptions under 500 characters for best appearance on all devices.

What if I downgrade from Pro to Lite?

Your products remain, but Pro-only features (custom colors, advanced layouts) revert to defaults. You can upgrade anytime without losing data.

Troubleshooting

Shortcode not displaying

Problem: Your shortcode appears as text instead of showing a cart.

Solution:

  1. Check that the plugin is activated in Plugins → Installed Plugins
  2. Verify the shortcode syntax: [affilicart id="225"] (where 225 is your cart ID number)
  3. Ensure you're editing in the block editor or classic editor, not a page builder (some builders don't support shortcodes)
  4. Clear your browser cache and reload the page

Amazon affiliate ID not saving

Problem: Settings page won't save your affiliate ID.

Solution:

  1. Double-check that your ID format is correct (usually ends with "-20")
  2. Clear browser cache and try again
  3. Check that you have Administrator access to your WordPress site (only admins can save plugin settings)
  4. Temporarily disable other plugins to check for conflicts

Cart appears broken on mobile

Problem: The cart looks stretched or misaligned on phones.

Solution:

  1. Affilicart is fully responsive—this may be a theme conflict
  2. Check your theme's CSS isn't interfering with cart styling
  3. Try switching to a default WordPress theme (like Twenty Twenty-Four) to test
  4. Check if other plugins are conflicting

Products aren't showing in search

Problem: Products don't appear in your WordPress site's search results.

Important: WordPress's native search function only searches standard post types like posts and pages. Affilicart products are a custom post type and don't appear in the standard WordPress search bar. This is by design — Affilicart products are managed through the Affilicart admin interface, not the WordPress post editor.

If you're trying to find products within Affilicart to add to carts, use the Affilicart → Products admin page where you can browse and manage all your products.

To make products searchable on your site: You can create regular WordPress posts or pages that recommend your products, and those will appear in site search. Use the shortcodes in those posts to display the actual Affilicart products.

Readers report wrong prices

Problem: Cart shows outdated or incorrect Amazon prices.

Solution:

Prices in Affilicart reflect what Amazon shows at the time of cart creation. Prices update dynamically when readers view the actual Amazon cart. This is normal behavior.
